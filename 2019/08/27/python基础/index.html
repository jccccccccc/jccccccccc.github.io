PYTHON基本数据类型

字符串

字符串就是一系列字符。在Python中，用引号括起的都是字符串，其中的引号可以是单引号也可以是双引号。

  • Python使用加号+来合并字符串
  • 要在字符串中添加制表符，可使用\t，添加换行符，可使用\n
  • lstrip()，rstrip()，strip()可删除字符串的空白

    数字

    在Python中，数字类型主要包括整数和浮点数。
  • 两个乘号**代表乘方运算
  • str()可将数字类型转换为字符串类型

列表

列表由一系列按特定顺序排列的元素组成。

  • 列表是可变的
  • 第一个列表元素的索引是[0]
  • append()可在列表末尾添加元素，insert(0,'a')可将字符串插至列表的第一个
  • del list[0]，pop(0)，remove('a')均可删除列表元素，但是方法remove()仅可删除第一个指定的值
  • sort()将列表永久排序，sorted()将列表临时排序，reverse()永久将列表倒序
  • for循环遍历整个列表
  • range()可创建数字列表，左闭右开
  • 切片是指处理列表的部分元素，复制列表时复制切片，不要直接复制列表

    元组

    不可变的列表称为元组。
  • 给元组变量赋值是合法的

    字典

    在Python中。字典是一系列键-值对，可将任何Python对象用作字典中的值。
  • 字典是一种动态结构，可随时在其中添加键-值对。
  • del dict['key']可删除字典元素
  • for key,value in dict.items()可遍历字典
  • keys()，values()代表字典的键和值
  • set()可去重
